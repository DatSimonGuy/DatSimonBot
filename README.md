# DatSimonBot v2.0.0
A telegram bot that makes student life easier and provides utility and games to a chat

## Changelog:
- Massive rewrite
- Add chat statistics and youtube functionality
- Removed some of the functions
- Balanced some aspects of the Contexto game
- Better performance

## Bot functions and features:

### Today's mission (NEW)
- Will send today's mission for fun - use /mission command to get it

### Group activity (NEW)
- /activity will display activity since last day, counts it at 00:00
- /activity all will display total activity and activity since last day
- /top will display chat memebers in order of who wrote the most messages
- /bottom will do the same but in reverse
- The bot won't display any member who didn't yet send any message since adding him to the group

### University plan management (CHANGED)
- You can add plans to bot and manage them
- It allows you to see who has free time right now so you can hang out
- For now lacks any gui, I will try to implement it with buttons later
- Meanwhile the parameters can be specified using --name, --start and so on

### Contexto game (CHANGED)
- Allows you to play contexto with friends on a groupchat
- It is now turn based

### Gif management
- DSB can store gifs under a tag which allows to reply to someone with a gif without searching through your entire collection of gifs.
- Multiple gifs can be stored under the same tag and the one sent is random among them.

### Stickers management
- Same as the gifs but with stickers
