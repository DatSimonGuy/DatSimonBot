add - /add <param> used to add gifs, stickers, lessons, majors and whatnot
remove - /remove <param> removes a thing of your choice
show - /show <param> shows you chosen data
edit - /edit <param> edits chosen data
plan - shows all available plans
join - /join <major> will join a major
link - use this to link private chats to the main group to share data
status - shows who is on a break right now

new_contexto - start new contexto game
c  - c <word> use this to guess the word
